# Inscripciones a Cursos - MySQL SKILLNEST

## Descripción

Este proyecto consiste en la creación de una base de datos relacional para gestionar las inscripciones de estudiantes a cursos utilizando MySQL.

El objetivo es practicar el diseño de modelos Entidad-Relación (ERD), la creación de tablas con claves primarias y foráneas, y la realización de consultas utilizando JOINs.

---

## Tecnologías Utilizadas

* MySQL Workbench
* Visual Studio Code (VSC)

---

## Aprendizajes Obtenidos

Durante el desarrollo de este proyecto se reforzaron conocimientos sobre:

* Diseño de bases de datos relacionales.
* Claves primarias y foráneas.
* Relaciones uno a muchos.
* Consultas JOIN.
* Funciones de agregación.
* Manipulación de datos mediante INSERT, SELECT y DELETE.
* Uso de MySQL Workbench para modelado y generación automática de scripts mediante Forward Engineer.
